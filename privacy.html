TodoList 隐私政策

生效日期： 2026年6月2日

最后更新： 2026年6月2日

1. 简介

TodoList（"我们"、"本应用"）是一款任务管理应用。我们高度重视您的隐私，本隐私政策旨在说明我们如何收集、使用、存储和保护您的个人信息。

请注意： 本应用主要处理本地数据，不强制要求用户注册账号或将数据上传至我们的服务器。数据同步功能仅在用户主动配置第三方服务时才会启用。

2. 我们收集的数据

2.1 您主动提供的数据

任务数据：

  • 任务标题、描述、优先级、截止日期
  • 任务分类名称和颜色
  • 子任务关系
  • 自定义排序顺序
  • 提醒设置（时间、重复规则）

账户数据（可选）：

  • 如果您使用第三方同步服务（Supabase），您的 API 地址和密钥仅存储在本地设备上
  • 我们不收集或存储您的第三方服务登录凭据

2.2 自动收集的数据

设备标识符：

  • 设备 ID（用于多端同步时标识设备来源）
  • 此 ID 是匿名化的，不关联到您的个人身份

使用数据：

  • 应用崩溃日志（用于改进应用稳定性）
  • 操作时间戳（用于数据同步的冲突解决）

我们不收集：

  • ❌ 地理位置信息
  • ❌ 联系人信息
  • ❌ 设备上的其他应用数据
  • ❌ 浏览历史或网络活动
  • ❌ 广告标识符（IDFA/AAID）
  • ❌ 个人身份信息（姓名、邮箱、电话等）

3. 数据存储

3.1 本地存储

所有任务数据默认存储在您的设备本地数据库中（SQLite）。我们不会将这些数据上传到我们的服务器。

3.2 同步数据

如果您启用了同步功能：

  • Supabase 同步： 数据存储在您自己配置的 Supabase 项目中，我们无权访问
  • 自定义 API 同步： 数据存储在您自己部署的服务器上

3.3 备份数据

应用支持创建本地备份文件，备份文件存储在设备本地存储中。您可以选择通过系统分享功能将备份文件导出到您选择的位置（如云盘、邮件等）。

4. 数据使用

我们使用您的数据仅限于以下目的：

  1. 提供核心功能： 任务管理、提醒、分类、排序
  2. 多端同步： 在您的多个设备之间同步任务状态
  3. 冲突解决： 当多端同时修改同一任务时，根据时间戳自动合并
  4. 应用改进： 匿名化的崩溃报告帮助我们修复 bug

我们不会：

  • 出售您的数据给第三方
  • 使用您的数据进行广告投放
  • 将您的数据用于人工智能训练
  • 超出上述目的使用您的数据

5. 数据安全

5.1 传输安全

  • 所有网络通信默认使用 HTTPS/TLS 加密
  • 如果您启用"允许不安全连接"设置（仅用于内网开发/测试），HTTP 通信将不受加密保护

5.2 本地安全

  • 数据库密码（如配置）仅用于数据库初始化，可随时清除
  • 应用使用操作系统提供的标准文件系统权限保护数据

5.3 诊断导出

  • 同步诊断和错误诊断导出功能会自动脱敏
  • 导出的文件中不包含任务标题、描述等原始内容
  • 仅包含 ID、状态、hash、时间戳等元数据

6. 数据保留与删除

6.1 保留期限

  • 活跃数据： 永久保留，直至您主动删除
  • 已删除任务： 保留 30 天后从数据库中移除（软删除机制）
  • 操作日志： 保留 90 天，用于同步冲突排查
  • 备份文件： 最多保留 5 个历史版本，旧版本自动清理

6.2 您的删除权利

您可以随时：

  • 删除单个任务或所有任务
  • 清除应用数据（通过系统设置）
  • 卸载应用（将删除所有本地数据）
  • 撤销同步服务的授权

7. 第三方服务

本应用可能使用以下第三方服务（仅在您主动配置时）：

服务 用途 数据类型
Supabase 数据同步 任务数据、设备 ID
自定义 API 数据同步 任务数据、设备 ID

这些第三方服务有其独立的隐私政策，我们建议您查阅相关服务提供商的隐私政策。

8. 儿童隐私

本应用不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。如果您发现儿童向我们提供了个人信息，请联系我们，我们将尽快删除相关信息。

9. 隐私政策变更

我们可能会不时更新本隐私政策。任何变更将在本页面发布，并更新"最后更新"日期。重大变更将通过应用内通知告知您。

10. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • 邮箱： privacy@xdolist.com
  • GitHub Issues： https://github.com/xdolist/todolist/issues

11. 权限说明

本应用需要以下系统权限：

  • 通知权限： 发送任务提醒通知（可选）
  • 网络权限： 同步数据到您配置的第三方服务（仅在启用同步时需要）
  • 本地网络权限（iOS）： 连接局域网内的自定义同步服务器（可选）

所有权限均为可选，拒绝授权不会影响应用的核心本地功能。

再次声明： TodoList 是一款尊重隐私的应用。您的数据属于您，我们仅提供工具帮助您管理数据，不占有、不分析、不出售您的个人信息。
